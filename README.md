# File Creator - консольная утилита для создания json-списка с контрольными суммами.

## Возможности
- Добавление одного файла (вычисление CRC32).
- Добавление всех файлов из директории (нерекурсивно или рекурсивно с флагом `-r`).
- Удаление файла из списка.
- Задание пути для сохранения JSON-списка (по умолчанию `file_list.json` в папке с утилитой).
- Сохранение списка в JSON-файл.
